OpenThai AI - YouTube Selling Roadmap

คู่มือขายของบน YouTube ตั้งแต่สมัครจนรับเงิน (73 ขั้นตอน)


ส่วนที่ 1 — สมัครบัญชี (7 ขั้น)

• สร้างบัญชี Google/YouTube

• ยืนยัน OTP

• ตั้งชื่อ

• สร้าง Channel

• ตั้งชื่อ Channel

• เปิด Notification


ส่วนที่ 2 — ตั้งค่า Channel (8 ขั้น)

• Profile / Banner / Description / Links / Trailer / Playlist / Keywords / Upload Defaults


ส่วนที่ 3 — YouTube Partner Program (6 ขั้น)

• 1,000 Subs + 4,000 Watch Hours → สมัคร YPP → AdSense → Approve


ส่วนที่ 4 — YouTube Shopping (5 ขั้น)

• เชื่อมร้านค้า → เพิ่มสินค้า → Tag ในวิดีโอ → Store Tab


ส่วนที่ 5 — วิดีโอยาว Long-form (9 ขั้น) [AI]

• [AI] Keyword Research → Script → ถ่าย → ตัดต่อ → Thumbnail → Title → Description → Tags → End Screen


ส่วนที่ 6 — YouTube Shorts (6 ขั้น) [AI]

• ถ่าย 9:16 → ตัดต่อ → เสียง Trending → [AI] Title+Hashtag → Tag สินค้า → โพสต์ 1-3/วัน


ส่วนที่ 7 — YouTube Live (6 ขั้น)

• Go Live → เชื่อมสินค้า → Live 30-60 นาที → Pin → Super Chat → บันทึก


ส่วนที่ 8 — YouTube SEO (5 ขั้น) [AI]

• [AI] Keyword Research → ใส่ Keyword → Analytics → ปรับ Thumbnail → Content ตาม Trending


ส่วนที่ 9 — YouTube Ads (5 ขั้น)

• Google Ads → Campaign → Ad Format → Target → งบ+ติดตาม


ส่วนที่ 10 — รับออร์เดอร์ (6 ขั้น)

• ลิงก์ Description → CTA → Pinned Comment → DM → ส่งไป LINE/WhatsApp → ชำระเงิน


ส่วนที่ 11 — รายได้ YouTube (5 ขั้น)

• Ad Revenue + Super Chat + Memberships + Shopping Affiliate + Sponsorship


ส่วนที่ 12 — Community (5 ขั้น)

• ขอรีวิว → ตอบ Comment → Community Tab → Revenue Analytics → ปรับ Strategy

- รวม 73 ขั้นตอน ใน 12 ส่วน - YouTube = Video Platform อันดับ 1 โลก 2,500M+ Users - หลายช่องทางรายได้: Ads + Super Chat + Membership + Shopping + Sponsorship - วิดีโอเก่ายังสร้างรายได้ต่อเนื่อง (Passive Income) - OpenThai AI ช่วยเขียน Script + Title + Description + Tags + Keyword Research
