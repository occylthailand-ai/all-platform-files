62 ขั้นตอน ใน 10 ส่วน | 2,000M+ Users ทั่วโลก | Catalog + Chat + Broadcast ครบจบ

ส่วนที่ 1 — สมัครและตั้งค่าบัญชี WhatsApp (6 ขั้น)

  1. โหลดแอป WhatsApp จาก Play Store / App Store
  2. ใส่เบอร์มือถือ (รองรับเบอร์ไทย +66)
  3. ยืนยัน OTP ทาง SMS หรือโทรศัพท์
  4. ตั้งชื่อโปรไฟล์ + รูปภาพที่สื่อถึงแบรนด์
  5. เขียน About 139 ตัวอักษร: ขายอะไร / ช่องทางสั่งซื้อ
  6. เปิด Notification รับแจ้งเตือนข้อความ

ส่วนที่ 2 — Switch to WhatsApp Business (7 ขั้น)

  1. โหลดแอป WhatsApp Business
  2. ย้ายบัญชีจาก WhatsApp ปกติ → Business
  3. ยืนยัน OTP เบอร์มือถือ
  4. กรอก Business Profile: ชื่อร้าน / หมวด / คำอธิบาย
  5. ใส่ข้อมูลติดต่อ: ที่อยู่ / อีเมล / เว็บไซต์
  6. ตั้งเวลาทำการ (Business Hours)
  7. เลือกหมวดธุรกิจ

ส่วนที่ 3 — ตั้งค่า Business Tools (7 ขั้น)

  1. ตั้ง Greeting Message: ข้อความต้อนรับอัตโนมัติ
  2. ตั้ง Away Message: ตอบกลับนอกเวลา
  3. สร้าง Quick Replies: ข้อความสำเร็จรูป (ราคา / วิธีสั่ง / ค่าส่ง)
  4. ตั้ง Labels (แท็กสี): จัดกลุ่มแชทลูกค้า
  5. สร้าง Short Link: wa.me/66XXXXXXXXX
  6. สร้าง QR Code ให้ลูกค้าสแกน
  7. เชื่อม WhatsApp กับ Facebook Page / Instagram

ส่วนที่ 4 — สร้าง WhatsApp Catalog (7 ขั้น) 🤖

  1. Business Tools → Catalog → เพิ่มสินค้า
  2. อัปโหลดรูปสินค้า สูงสุด 10 รูปต่อสินค้า
  3. ตั้งชื่อสินค้า + ราคา
  4. 🤖 เขียนรายละเอียดสินค้า: วัสดุ / ขนาด / วิธีใช้ / จุดเด่น
  5. ใส่ลิงก์ซื้อสินค้า (เว็บ / Shopee / Lazada)
  6. จัดกลุ่มสินค้าเป็น Collection
  7. แชร์ Catalog Link ให้ลูกค้าเข้าชม

ส่วนที่ 5 — สร้างคอนเทนต์และโปรโมท (7 ขั้น) 🤖

  1. 🤖 โพสต์ WhatsApp Status: รูป / วิดีโอ / ข้อความ (อายุ 24 ชม.)
  2. โพสต์สินค้าใน Status 3–5 ครั้ง/วัน
  3. 🤖 สร้าง Broadcast List: ส่งข้อความถึงลูกค้าหลายคนพร้อมกัน
  4. 🤖 ส่ง Broadcast โปรโมชั่น / สินค้าใหม่ / Flash Sale
  5. สร้าง WhatsApp Group ลูกค้า VIP
  6. แชร์ลิงก์ wa.me/ และ QR Code บน TikTok / FB / LINE / IG
  7. ใช้ WhatsApp Channel: โพสต์ข่าวสารถึง Follower ไม่จำกัด

ส่วนที่ 6 — รับออร์เดอร์และสื่อสาร (7 ขั้น)

  1. รับข้อความสั่งซื้อจากลูกค้าใน Chat
  2. ส่ง Catalog สินค้าให้ลูกค้าเลือกใน Chat
  3. ลูกค้ากด "เพิ่มลงตะกร้า" → ส่ง Cart มาใน Chat
  4. ตอบคำถามสินค้า / ราคา / วิธีสั่ง
  5. ยืนยันออร์เดอร์: สินค้า / จำนวน / ที่อยู่
  6. แจ้งราคารวม: ค่าสินค้า + ค่าส่ง
  7. ส่ง QR PromptPay / PayPal / Wise ให้ลูกค้า

ส่วนที่ 7 — ตรวจสอบชำระเงิน (5 ขั้น)

  1. รอลูกค้าโอนเงินผ่าน PromptPay / PayPal / Wise
  2. ตรวจสอบสลิปหรือยอดเข้าใน Mobile Banking
  3. แจ้งลูกค้า "ยืนยันชำระเงินแล้ว" ทาง Chat
  4. เปลี่ยน Label แชทเป็น "ชำระแล้ว"
  5. บันทึกออร์เดอร์ลงระบบ

ส่วนที่ 8 — แพ็คและจัดส่ง (5 ขั้น)

  1. แพ็คสินค้า + ใบปะหน้าที่อยู่ลูกค้า
  2. ส่งพัสดุกับขนส่ง (Flash / Kerry / J&T / EMS ต่างประเทศ)
  3. แจ้งเลข Tracking ทาง WhatsApp Chat
  4. เปลี่ยน Label แชทเป็น "จัดส่งแล้ว"
  5. ลูกค้ารับสินค้า → ขอรีวิวทาง Chat

ส่วนที่ 9 — WhatsApp Business API สำหรับ Scale (5 ขั้น)

  1. สมัคร WhatsApp Business API ผ่าน Meta Business Suite
  2. เชื่อม API กับ CRM / Chatbot (Respond.io / WATI / Twilio)
  3. ตั้ง Automated Message Flows: Welcome → Order → Payment → Shipping
  4. ส่ง Template Messages: แจ้ง Tracking / ยืนยันออร์เดอร์อัตโนมัติ
  5. ใช้ Click-to-WhatsApp Ads จาก Facebook / Instagram

ส่วนที่ 10 — หลังการขายและ Retention (6 ขั้น)

  1. ขอรีวิวและรูปถ่ายจากลูกค้าทาง Chat
  2. แชร์รีวิวลูกค้าจริงใน WhatsApp Status
  3. Follow up ลูกค้าหลัง 3–5 วัน ทาง Chat
  4. เพิ่มลูกค้าเข้า VIP Group รับโปรโมชั่นพิเศษ
  5. ส่ง Broadcast สินค้าใหม่ / โปรโมชั่นประจำสัปดาห์
  6. วิเคราะห์ WhatsApp Business Statistics
