🇮🇩 Tokopedia — ขายของในอินโดนีเซียจนรับเงินครบ

65 ขั้นตอน ใน 11 ส่วน | 100M+ Users | Marketplace อันดับ 1 อินโดนีเซีย

ส่วนที่ 1 — สมัครบัญชี Tokopedia (6 ขั้น)

  1. โหลดแอป Tokopedia หรือเข้า tokopedia.com
  2. กด "Daftar" → ใส่เบอร์/อีเมล
  3. ยืนยัน OTP
  4. ตั้งชื่อบัญชีและรหัสผ่าน
  5. กรอกข้อมูลส่วนตัว
  6. เปิด Notification

ส่วนที่ 2 — เปิดร้านค้า (7 ขั้น)

  1. กด "Buka Toko" (เปิดร้าน)
  2. ตั้งชื่อร้านค้า
  3. เลือกหมวดหมู่สินค้าหลัก
  4. ใส่ที่อยู่ร้าน / Warehouse
  5. ใส่รูปโปรไฟล์ + ปก
  6. เขียน Shop Description
  7. ตั้ง Operation Hours

ส่วนที่ 3 — ยืนยันตัวตนและอัปเกรด (6 ขั้น)

  1. Verifikasi Toko → ยืนยันตัวตน
  2. อัปโหลด KTP / Passport
  3. ถ่ายรูป Selfie คู่เอกสาร
  4. รอ Tokopedia ยืนยัน 1–3 วัน
  5. อัปเกรดเป็น Power Merchant (Badge + ฟีเจอร์เพิ่ม)
  6. อัปเกรดเป็น Official Store (สำหรับแบรนด์)

ส่วนที่ 4 — ตั้งค่าการเงิน (5 ขั้น)

  1. Saldo → เพิ่มบัญชีธนาคาร (BCA/BRI/Mandiri/BNI)
  2. ใส่ชื่อ / เลขบัญชี / ธนาคาร
  3. ยืนยันบัญชีธนาคาร
  4. ตั้งค่า Saldo Tokopedia
  5. เปิด GoPay / OVO (ตัวเลือก)

ส่วนที่ 5 — ตั้งค่าการจัดส่ง (5 ขั้น)

  1. Pengiriman → เลือกขนส่ง
  2. เปิด JNE / J&T / SiCepat / AnterAja / GoSend / GrabExpress
  3. ตั้ง Pickup Address
  4. ตั้งน้ำหนัก/ขนาด Default
  5. ตั้ง Free Shipping Threshold

ส่วนที่ 6 — เพิ่มสินค้า (9 ขั้น) 🤖

  1. Tambah Produk (เพิ่มสินค้า)
  2. เลือกหมวดหมู่ (Kategori)
  3. 🤖 ตั้งชื่อสินค้าภาษาอินโดนีเซีย
  4. อัปโหลดรูป 2–9 รูป
  5. อัปโหลดวิดีโอ
  6. 🤖 เขียนรายละเอียดภาษาอินโดนีเซีย
  7. ตั้งราคา IDR + ราคาโปรโมชั่น
  8. ตั้งสต็อก + Variant
  9. ตั้งน้ำหนักสินค้า

🤖 OpenThai AI ช่วยเขียนชื่อสินค้า + รายละเอียดภาษาอินโดนีเซียได้

ส่วนที่ 7 — โปรโมชั่นและ Voucher (6 ขั้น)

  1. Iklan & Promosi → สร้างโปรโมชั่น
  2. สร้าง Voucher Toko: ส่วนลด / ฟรีค่าส่ง
  3. ตั้ง Flash Sale
  4. เข้าร่วม Campaign: WIB / Ramadan / 11.11 / 12.12
  5. ตั้ง Bundling
  6. ใช้ Broadcast Chat

ส่วนที่ 8 — TopAds (5 ขั้น)

  1. TopAds → สร้างแคมเปญ
  2. เลือก Product / Shop / Display Ads
  3. เลือกสินค้า + Keyword
  4. ตั้งงบ + Bid per Click
  5. ติดตาม CTR / CPC / ROAS

ส่วนที่ 9 — รับออร์เดอร์ (6 ขั้น)

  1. รับ Notification ออร์เดอร์ใหม่
  2. ตรวจสอบรายละเอียด
  3. กด "Terima Pesanan" ภายใน 2 วัน
  4. แพ็คสินค้า
  5. พิมพ์ใบปะหน้าพัสดุ
  6. ส่งพัสดุ / Request Pickup

ส่วนที่ 10 — ชำระเงินและรับเงิน (5 ขั้น)

  1. ลูกค้าชำระ GoPay / OVO / VA / บัตรเครดิต / Alfamart / Indomaret
  2. เงินเข้า Escrow (Rekening Bersama)
  3. ลูกค้ากด "Selesai" → เงิน Release
  4. กด "Tarik Dana" ถอนไปบัญชีธนาคาร
  5. รับเงินภายใน 1–2 วันทำการ

ส่วนที่ 11 — หลังการขายและรีวิว (5 ขั้น)

  1. ติดตามสถานะพัสดุ
  2. จัดการ Komplain / Return / Refund
  3. ตอบรีวิว (Ulasan) ทุกรายการ
  4. ขอรีวิวจากลูกค้าผ่าน Chat
  5. วิเคราะห์ Dashboard: ยอดขาย / Conversion / Traffic
