ส่วนที่ 1 — สมัครและตั้งค่าบัญชี Naver (7 ขั้น)

  1. ไปที่ naver.com → กด "회원가입" (Register)
  2. ใส่อีเมลหรือเบอร์มือถือ (รองรับเบอร์ต่างประเทศ)
  3. ยืนยัน OTP ทางเบอร์มือถือหรืออีเมล
  4. ตั้ง Naver ID + รหัสผ่าน
  5. กรอกข้อมูลส่วนตัว: ชื่อ / วันเกิด
  6. ใส่รูปโปรไฟล์ที่สื่อถึงแบรนด์
  7. เปิด Notification รับแจ้งเตือนข้อความและออร์เดอร์

ส่วนที่ 2 — สมัครเปิด Naver Smart Store (8 ขั้น)

  1. ไปที่ sell.smartstore.naver.com → กด "입점 신청"
  2. เลือกประเภทผู้ขาย: บุคคล / นิติบุคคล / ผู้ขายต่างชาติ
  3. กรอกข้อมูลธุรกิจ: ชื่อร้าน / ประเภทสินค้า / ที่อยู่
  4. อัปโหลดเอกสาร: ทะเบียนพาณิชย์ / Passport
  5. เชื่อมบัญชีธนาคารเกาหลี หรือ Payoneer
  6. ลงนามสัญญาผู้ขายออนไลน์
  7. รอ Naver Approve 3–7 วันทำการ
  8. ได้รับ Smart Store Account เข้า Seller Dashboard

ส่วนที่ 3 — ตั้งค่าร้านค้า Smart Store (6 ขั้น)

  1. ตั้งชื่อร้านภาษาเกาหลี + โลโก้ร้านค้า
  2. ออกแบบหน้าร้าน: Banner / Theme / Layout
  3. เขียน Shop Description ภาษาเกาหลี
  4. ตั้งค่าการจัดส่ง: CJ Logistics / Hanjin / Lotte + ค่าส่ง
  5. เชื่อม Naver Pay สำหรับรับชำระเงิน
  6. ตั้ง Return / Exchange Policy ตามมาตรฐานเกาหลี

ส่วนที่ 4 — เพิ่มสินค้าและตั้งค่า Listing (9 ขั้น) 🤖

  1. Dashboard → สินค้า → เพิ่มสินค้าใหม่
  2. เลือกหมวดหมู่สินค้าให้ถูกต้อง
  3. ตั้งชื่อสินค้าภาษาเกาหลี: Keyword + จุดเด่น 🤖
  4. อัปโหลดรูปสินค้า: Main Image + รูปรายละเอียด 3–10 รูป
  5. สร้าง Detail Page (상세페이지): Infographic สวยงาม
  6. เขียนรายละเอียดสินค้าภาษาเกาหลี 🤖
  7. ตั้งราคา KRW + ราคาโปรโมชั่น
  8. ตั้งสต็อก + SKU Variant
  9. Submit สินค้ารอ Approve 24–48 ชั่วโมง

ส่วนที่ 5 — Naver SEO (6 ขั้น) 🤖

  1. วิจัย Keyword ภาษาเกาหลีเกี่ยวกับสินค้าไทย 🤖
  2. ใส่ Keyword ใน Product Title, Description, Tags
  3. สร้าง Naver Blog เขียนรีวิวสินค้าภาษาเกาหลี 🤖
  4. โพสต์ใน Naver Cafe ชุมชนเฉพาะกลุ่ม
  5. สร้างคอนเทนต์ใน Naver Post บทความยาว 🤖
  6. ติดตาม Search Ranking ใน Naver Webmaster Tools

ส่วนที่ 6 — โปรโมชั่นและ Marketing (6 ขั้น)

  1. สร้าง Coupon ส่วนลด: % หรือลดราคาคงที่
  2. ตั้ง Time Deal: โปรลดราคาจำกัดเวลา
  3. เข้าร่วม Naver Shopping Campaign: 11.11 / ฤดูกาล
  4. ตั้ง Bundle Deal: ซื้อหลายชิ้นราคาถูก
  5. ใช้ Naver Shopping Live ขายของสด
  6. ให้ Naver Point / Shopping Cash กับลูกค้า

ส่วนที่ 7 — Naver Ads (5 ขั้น)

  1. เข้า searchad.naver.com → สร้าง Ads Account
  2. สร้าง Search Ads Campaign เลือก Keyword
  3. ตั้ง Bid ราคาประมูลต่อ Keyword
  4. ตั้งงบโฆษณาต่อวัน
  5. ติดตาม CTR / CPC / ROAS / Conversion

ส่วนที่ 8 — Naver Blog และ Cafe สร้าง Trust (5 ขั้น) 🤖

  1. สร้าง Naver Blog ของแบรนด์ เขียนรีวิวภาษาเกาหลี 🤖
  2. โพสต์ Blog 2–3 ครั้ง/สัปดาห์ ใส่ Keyword SEO 🤖
  3. เข้าร่วม Naver Cafe ที่เกี่ยวข้อง
  4. ขอ Naver Influencer (블로거) รีวิวสินค้า
  5. ใส่ลิงก์ Smart Store ในทุก Blog Post

ส่วนที่ 9 — รับออร์เดอร์และจัดการ (6 ขั้น)

  1. รับ Notification ออร์เดอร์ใหม่ใน Seller Dashboard
  2. ตรวจสอบรายละเอียด: สินค้า / จำนวน / ที่อยู่ / Naver Pay
  3. ยืนยันออร์เดอร์และเตรียมสินค้าภายใน 48 ชั่วโมง
  4. แพ็คสินค้าตาม Standard ของ Naver
  5. พิมพ์ใบปะหน้าพัสดุจาก Dashboard
  6. ส่งพัสดุกับขนส่ง CJ / Hanjin หรือ EMS จากไทย

ส่วนที่ 10 — ชำระเงินและรับเงิน (5 ขั้น)

  1. ลูกค้าชำระผ่าน Naver Pay / บัตรเครดิต / โอนธนาคาร
  2. เงินเข้า Naver Escrow รอ Release หลังลูกค้ารับสินค้า
  3. เงิน Release เข้า Smart Store Wallet
  4. โอนเงินจาก Wallet ไปบัญชีธนาคาร / Payoneer
  5. แลก KRW → THB ผ่านธนาคารหรือ Payoneer

ส่วนที่ 11 — หลังการขายและรีวิว (6 ขั้น)

  1. ติดตามสถานะพัสดุใน Dashboard
  2. จัดการ Return / Exchange ตามนโยบาย
  3. ตอบรีวิวลูกค้าทุกรายการ (สำคัญมากในเกาหลี)
  4. ขอรีวิวจากลูกค้าหลังรับสินค้า
  5. วิเคราะห์ Store Analytics: ยอดขาย / Conversion / Traffic
  6. ปรับปรุง Listing และ Blog Content ตาม Performance
