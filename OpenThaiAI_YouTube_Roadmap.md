# OpenThai AI - YouTube Selling Roadmap
## คู่มือขายของบน YouTube ตั้งแต่สมัครจนรับเงิน (73 ขั้นตอน)

## ส่วนที่ 1 — สมัครบัญชี (7 ขั้น)
1. สร้างบัญชี Google/YouTube
2. ยืนยัน OTP
3. ตั้งชื่อ
4. สร้าง Channel
5. ตั้งชื่อ Channel
6-7. เปิด Notification

## ส่วนที่ 2 — ตั้งค่า Channel (8 ขั้น)
8-15. Profile / Banner / Description / Links / Trailer / Playlist / Keywords / Upload Defaults

## ส่วนที่ 3 — YouTube Partner Program (6 ขั้น)
16-21. 1,000 Subs + 4,000 Watch Hours → สมัคร YPP → AdSense → Approve

## ส่วนที่ 4 — YouTube Shopping (5 ขั้น)
22-26. เชื่อมร้านค้า → เพิ่มสินค้า → Tag ในวิดีโอ → Store Tab

## ส่วนที่ 5 — วิดีโอยาว Long-form (9 ขั้น) [AI]
27-35. [AI] Keyword Research → Script → ถ่าย → ตัดต่อ → Thumbnail → Title → Description → Tags → End Screen

## ส่วนที่ 6 — YouTube Shorts (6 ขั้น) [AI]
36-41. ถ่าย 9:16 → ตัดต่อ → เสียง Trending → [AI] Title+Hashtag → Tag สินค้า → โพสต์ 1-3/วัน

## ส่วนที่ 7 — YouTube Live (6 ขั้น)
42-47. Go Live → เชื่อมสินค้า → Live 30-60 นาที → Pin → Super Chat → บันทึก

## ส่วนที่ 8 — YouTube SEO (5 ขั้น) [AI]
48-52. [AI] Keyword Research → ใส่ Keyword → Analytics → ปรับ Thumbnail → Content ตาม Trending

## ส่วนที่ 9 — YouTube Ads (5 ขั้น)
53-57. Google Ads → Campaign → Ad Format → Target → งบ+ติดตาม

## ส่วนที่ 10 — รับออร์เดอร์ (6 ขั้น)
58-63. ลิงก์ Description → CTA → Pinned Comment → DM → ส่งไป LINE/WhatsApp → ชำระเงิน

## ส่วนที่ 11 — รายได้ YouTube (5 ขั้น)
64-68. Ad Revenue + Super Chat + Memberships + Shopping Affiliate + Sponsorship

## ส่วนที่ 12 — Community (5 ขั้น)
69-73. ขอรีวิว → ตอบ Comment → Community Tab → Revenue Analytics → ปรับ Strategy
