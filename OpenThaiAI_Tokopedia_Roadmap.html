OpenThai AI - Tokopedia Selling Roadmap

คู่มือขายของบน Tokopedia ตั้งแต่สมัครจนรับเงิน (65 ขั้นตอน)


ส่วนที่ 1 — สมัครบัญชี (6 ขั้น)

1-6. โหลดแอป → Daftar → OTP → ชื่อ → ข้อมูล → Notification

ส่วนที่ 2 — เปิดร้าน (7 ขั้น)

7-13. Buka Toko → ชื่อร้าน → หมวด → ที่อยู่ → รูป → Description → เวลาทำการ

ส่วนที่ 3 — ยืนยันตัวตน (6 ขั้น)

14-19. Verifikasi → KTP/Passport → Selfie → รอ Approve → Power Merchant → Official Store

ส่วนที่ 4 — การเงิน (5 ขั้น)

20-24. บัญชีธนาคาร BCA/BRI/Mandiri → ยืนยัน → Saldo → GoPay/OVO

ส่วนที่ 5 — จัดส่ง (5 ขั้น)

25-29. เลือกขนส่ง JNE/J&T/SiCepat → Pickup Address → น้ำหนัก → Free Shipping

ส่วนที่ 6 — เพิ่มสินค้า (9 ขั้น) [AI]

30-38. Tambah Produk → Kategori → [AI] ชื่อภาษาอินโดฯ → รูป → วิดีโอ → [AI] รายละเอียด → ราคา IDR → Variant → น้ำหนัก

ส่วนที่ 7 — โปรโมชั่น (6 ขั้น)

39-44. Iklan & Promosi → Voucher → Flash Sale → Campaign WIB/11.11 → Bundling → Broadcast Chat

ส่วนที่ 8 — TopAds (5 ขั้น)

45-49. สร้างแคมเปญ → Product/Shop/Display → Keyword → งบ+Bid → ติดตาม CTR/ROAS

ส่วนที่ 9 — รับออร์เดอร์ (6 ขั้น)

50-55. Notification → ตรวจสอบ → Terima Pesanan → แพ็ค → ใบปะหน้า → ส่ง/Pickup

ส่วนที่ 10 — รับเงิน (5 ขั้น)

56-60. GoPay/OVO/VA/Alfamart → Escrow → Selesai → Tarik Dana → รับเงิน 1-2 วัน

ส่วนที่ 11 — หลังการขาย (5 ขั้น)

61-65. ติดตามพัสดุ → Komplain/Return → ตอบ Ulasan → ขอรีวิว → Analytics
