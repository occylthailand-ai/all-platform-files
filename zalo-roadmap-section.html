🇻🇳 Zalo — ขายของในเวียดนามจนรับเงินครบ

65 ขั้นตอน ใน 11 ส่วน | 75M+ Users | OA + Shop + ZaloPay ครบจบ

ส่วนที่ 1 — สมัครบัญชี Zalo (7 ขั้น)

  1. โหลดแอป Zalo จาก Play Store / App Store
  2. กด "สมัคร" → ใส่เบอร์มือถือ
  3. ยืนยัน OTP ทาง SMS
  4. ตั้งชื่อโปรไฟล์ที่สื่อถึงแบรนด์
  5. ใส่รูปโปรไฟล์ + รูปปก
  6. ตั้งรหัสผ่าน
  7. เปิด Notification

ส่วนที่ 2 — สมัคร Zalo Official Account (7 ขั้น)

  1. ไปที่ oa.zalo.me → กด "Tạo OA"
  2. ลงทะเบียนด้วยบัญชี Zalo
  3. เลือกประเภท OA: ร้านค้า / แบรนด์ / บริการ
  4. กรอกข้อมูลธุรกิจ
  5. อัปโหลดเอกสาร: ทะเบียนพาณิชย์ / Passport
  6. รอ Zalo อนุมัติ 1–3 วัน
  7. ได้รับ Official Account พร้อมใช้งาน

ส่วนที่ 3 — ตั้งค่า OA และโปรไฟล์ (6 ขั้น)

  1. ตั้งชื่อ OA + โลโก้ + รูปปก
  2. เขียน OA Description ภาษาเวียดนาม
  3. ตั้งค่า Auto Reply: ข้อความต้อนรับ
  4. สร้าง Menu: ปุ่ม "สินค้า / สั่งซื้อ / ราคา / ติดต่อ"
  5. ตั้ง Quick Reply สำเร็จรูป
  6. เพิ่มข้อมูลติดต่อ: เบอร์ / อีเมล / เว็บ

ส่วนที่ 4 — ตั้งค่า Zalo Shop (7 ขั้น) 🤖

  1. OA Management → Shop → เปิดใช้งาน
  2. ตั้งชื่อร้านและโลโก้
  3. 🤖 เพิ่มสินค้า: ชื่อ / รายละเอียด / ราคา VND / รูป / สต็อก
  4. จัดกลุ่มสินค้าเป็น Category
  5. ตั้งค่าการจัดส่ง
  6. เชื่อม ZaloPay / บัญชีธนาคารเวียดนาม
  7. เปิด Shop Tab ใน OA

ส่วนที่ 5 — ตั้งค่า ZaloPay (5 ขั้น)

  1. โหลดแอป ZaloPay
  2. สมัคร ZaloPay ด้วยเบอร์เวียดนาม
  3. ยืนยันตัวตน (CCCD / Passport)
  4. ผูกบัญชีธนาคารเวียดนาม
  5. เปิดใช้งาน ZaloPay รับเงินจากลูกค้า

ส่วนที่ 6 — สร้างคอนเทนต์และโปรโมท (7 ขั้น) 🤖

  1. 🤖 สร้างโพสต์ใน Zalo OA: รูปสินค้า + ข้อความเวียดนาม
  2. โพสต์ใน Zalo Moment (Timeline)
  3. 🤖 ส่ง Broadcast Message หา Follower ทั้งหมด
  4. 🤖 สร้าง Zalo Article: บทความยาวใน OA
  5. สร้าง Voucher / Coupon ส่วนลด
  6. แชร์ QR Code OA บน Facebook / TikTok / IG
  7. ใช้ Zalo Ads โปรโมท OA และสินค้า

ส่วนที่ 7 — Zalo Ads (5 ขั้น)

  1. เข้า ads.zalo.me → สร้าง Ads Account
  2. สร้าง Campaign: เลือก Objective
  3. ตั้ง Target Audience: อายุ / เพศ / ที่อยู่ / ความสนใจ
  4. ตั้งงบโฆษณาต่อวัน
  5. เผยแพร่ + ติดตาม Reach / CTR / CPC

ส่วนที่ 8 — รับออร์เดอร์ (6 ขั้น)

  1. รับข้อความสั่งซื้อใน OA Chat
  2. ตอบคำถามสินค้า / ราคา / วิธีสั่ง
  3. ส่งสินค้าจาก Zalo Shop ให้ลูกค้าเลือกใน Chat
  4. ยืนยันออร์เดอร์: สินค้า / จำนวน / ที่อยู่
  5. แจ้งราคารวม: ค่าสินค้า + ค่าส่ง (VND)
  6. ส่ง QR ZaloPay / เลขบัญชีธนาคาร

ส่วนที่ 9 — ตรวจสอบชำระเงิน (5 ขั้น)

  1. ลูกค้าชำระผ่าน ZaloPay / โอนธนาคาร / COD
  2. ตรวจสอบยอดเข้าใน ZaloPay / Mobile Banking
  3. แจ้งลูกค้า "ยืนยันชำระเงินแล้ว"
  4. ติด Label "ชำระแล้ว"
  5. บันทึกออร์เดอร์

ส่วนที่ 10 — แพ็คและจัดส่ง (5 ขั้น)

  1. แพ็คสินค้า + ใบปะหน้า
  2. ส่งพัสดุ: GHN / Viettel Post / J&T VN / GHTK
  3. แจ้ง Tracking ทาง Zalo Chat
  4. ลูกค้าติดตามพัสดุ
  5. รับสินค้า → ขอรีวิว

ส่วนที่ 11 — หลังการขายและ Retention (5 ขั้น)

  1. ขอรีวิวและรูปถ่ายจากลูกค้า
  2. โพสต์รีวิวจริงใน OA / Moment
  3. Follow up หลัง 3–5 วัน
  4. ส่ง Broadcast โปรโมชั่นประจำสัปดาห์
  5. วิเคราะห์ OA Analytics: Followers / Messages / ยอดขาย
