OpenThai AI - Zalo Selling Roadmap

คู่มือขายของบน Zalo ตั้งแต่สมัครจนรับเงิน (65 ขั้นตอน)


ส่วนที่ 1 — สมัครบัญชี Zalo (7 ขั้น)

1-7. โหลดแอป → สมัคร → OTP → ชื่อ → รูป → รหัสผ่าน → Notification

ส่วนที่ 2 — สมัคร Zalo Official Account (7 ขั้น)

8-14. oa.zalo.me → ลงทะเบียน → เลือกประเภท → ข้อมูลธุรกิจ → เอกสาร → รอ Approve → ได้ OA

ส่วนที่ 3 — ตั้งค่า OA (6 ขั้น)

15-20. ชื่อ+โลโก้ → Description → Auto Reply → Menu → Quick Reply → ข้อมูลติดต่อ

ส่วนที่ 4 — Zalo Shop (7 ขั้น) [AI]

21-27. เปิด Shop → ชื่อร้าน → [AI] เพิ่มสินค้า → Category → จัดส่ง → ZaloPay → Shop Tab

ส่วนที่ 5 — ZaloPay (5 ขั้น)

28-32. โหลดแอป → สมัคร → ยืนยันตัวตน → ผูกธนาคาร → เปิดรับเงิน

ส่วนที่ 6 — Content และโปรโมท (7 ขั้น) [AI]

33-39. [AI] โพสต์ OA → Moment → [AI] Broadcast → [AI] Article → Voucher → QR Code → Ads

ส่วนที่ 7 — Zalo Ads (5 ขั้น)

40-44. สร้าง Account → Campaign → Target → งบ → ติดตาม

ส่วนที่ 8 — รับออร์เดอร์ (6 ขั้น)

45-50. รับข้อความ → ตอบคำถาม → ส่ง Shop → ยืนยัน → ราคา → QR ZaloPay

ส่วนที่ 9 — ชำระเงิน (5 ขั้น)

51-55. ZaloPay/โอน/COD → ตรวจสอบ → ยืนยัน → Label → บันทึก

ส่วนที่ 10 — จัดส่ง (5 ขั้น)

56-60. แพ็ค → GHN/Viettel Post/J&T → Tracking → ติดตาม → ขอรีวิว

ส่วนที่ 11 — หลังการขาย (5 ขั้น)

61-65. รีวิว → โพสต์รีวิว → Follow up → Broadcast → Analytics

สรุป

- รวม 65 ขั้นตอน ใน 11 ส่วน - Zalo = Super App อันดับ 1 เวียดนาม 75M+ Users - ตลาดเวียดนาม E-commerce โตเร็วอันดับ 1 อาเซียน - COD นิยมมาก (เก็บเงินปลายทาง) - OA ฟรี ไม่มีค่า Subscription - OpenThai AI ช่วยเขียน Post + Broadcast + Article ภาษาเวียดนามได้
