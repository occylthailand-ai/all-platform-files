🐦‍⬛ X (Twitter) — ขายของจนรับเงินครบ

63 ขั้นตอน ใน 11 ส่วน | Real-time Content → Trending → ขายได้ทั่วโลก

ส่วนที่ 1 — สมัครและตั้งค่าบัญชี X (7 ขั้น)

  1. โหลดแอป X จาก Play Store / App Store หรือเข้า x.com
  2. กด "สร้างบัญชี" → ใส่ชื่อ + อีเมลหรือเบอร์มือถือ
  3. ยืนยัน OTP ทางอีเมลหรือเบอร์มือถือ
  4. ตั้ง Username (@handle) ที่สื่อถึงแบรนด์ สั้น จำง่าย
  5. ตั้งรหัสผ่านอย่างน้อย 8 ตัวอักษร
  6. ใส่รูปโปรไฟล์ที่สื่อถึงแบรนด์หรือสินค้า
  7. เปิด Notification รับแจ้งเตือน DM และ Mention

ส่วนที่ 2 — ตั้งค่าโปรไฟล์ให้พร้อมขาย (7 ขั้น)

  1. ใส่รูปปก (Header Image) 1500x500px แสดงสินค้าหรือโปรโมชั่น
  2. เขียน Bio 160 ตัวอักษร: ขายอะไร / จุดเด่น / ช่องทางสั่งซื้อ
  3. ใส่ลิงก์เว็บไซต์ / LINE / Shopee / TikTok Shop ใน Profile
  4. ใส่ Location: ประเทศไทย หรือจังหวัดที่อยู่
  5. สมัคร X Premium เพื่อได้ Verified Badge ✓ ($8/เดือน)
  6. เปิด Professional Account: Settings → Professional Tools
  7. เลือกหมวดธุรกิจ: Shopping / Retail / Food / Beauty ฯลฯ

ส่วนที่ 3 — ตั้งค่า X Shopping (4 ขั้น)

  1. ไปที่ Professional Tools → Shopping → เปิดใช้งาน X Shops
  2. เชื่อมร้านค้าภายนอก: Shopify / Shopee / เว็บไซต์
  3. เพิ่มสินค้าใน X Shop: ชื่อ / รูป / ราคา / ลิงก์ซื้อ
  4. เปิด Shop Tab ใน Profile ให้ลูกค้าเข้าชมสินค้า

⚠️ X Shopping ยังไม่เปิดทุกประเทศ — ใช้ลิงก์ในโพสต์แทนได้เลย

ส่วนที่ 4 — สร้างคอนเทนต์โพสต์ขาย (9 ขั้น) 🤖

  1. เขียนโพสต์ (Tweet) สูงสุด 280 ตัวอักษร (Premium ได้ 25,000)
  2. แนบรูปสินค้า สูงสุด 4 รูปต่อโพสต์
  3. แนบวิดีโอสินค้า สูงสุด 2 นาที 20 วิ
  4. ใส่ Hashtag (#) 2–5 อัน ที่ Trending หรือเกี่ยวข้อง 🤖
  5. ใส่ลิงก์ร้านค้า / สินค้า / LINE ในโพสต์
  6. สร้าง Thread: โพสต์ต่อเนื่องเล่าเรื่องสินค้าละเอียด 🤖
  7. ใช้ Poll สอบถามความคิดเห็นลูกค้า ดึง Engagement
  8. โพสต์สม่ำเสมอ 3–5 ครั้ง/วัน 🤖
  9. Pin โพสต์ขายดีหรือโปรโมชั่นไว้ด้านบน Profile

🤖 OpenThai AI ช่วยเขียน Tweet + Thread + Hashtag + Caption ได้

ส่วนที่ 5 — สร้าง Engagement (6 ขั้น)

  1. Reply ทุก Comment / Mention จากลูกค้า
  2. Repost คอนเทนต์ที่เกี่ยวข้องกับสินค้า
  3. Quote Repost โพสต์ตัวเองพร้อม Caption ใหม่
  4. Like และ Reply โพสต์ของ Influencer ในกลุ่มเดียวกัน
  5. เข้าร่วมสนทนา Trending Topics ที่เกี่ยวข้อง
  6. สร้าง X Spaces: พูดคุยสดเรื่องสินค้า / Q&A

ส่วนที่ 6 — X Ads โปรโมทสินค้า (5 ขั้น)

  1. ไปที่ ads.x.com → สร้าง Ads Account
  2. สร้าง Campaign: เลือก Objective
  3. ตั้ง Target Audience: อายุ / เพศ / ความสนใจ / ภูมิภาค
  4. ตั้งงบโฆษณา Daily Budget + Bid Strategy
  5. เผยแพร่โฆษณาและติดตาม CTR / CPC ใน Dashboard

ส่วนที่ 7 — ดึง Traffic ไปร้านค้า (5 ขั้น)

  1. ใส่ลิงก์ Shopee / TikTok Shop / LINE / เว็บ ในทุกโพสต์ขาย
  2. ใช้ Link Shortener (Bitly) ติดตาม Click
  3. บอกให้ลูกค้า DM เพื่อสั่งซื้อโดยตรง
  4. แชร์โพสต์ X ไปยัง FB / LINE / IG ขยาย Reach
  5. สร้าง X List รวม Influencer / ลูกค้า / คู่ค้า

ส่วนที่ 8 — รับออร์เดอร์ (6 ขั้น)

  1. รับ DM คำสั่งซื้อจากลูกค้าใน X
  2. ตอบคำถามสินค้า / ราคา / วิธีสั่งซื้อใน DM
  3. ส่งรูปสินค้าเพิ่มเติมหรือวิดีโอรีวิวใน DM
  4. ยืนยันออร์เดอร์: สินค้า / จำนวน / ที่อยู่จัดส่ง
  5. แจ้งราคารวม: ค่าสินค้า + ค่าจัดส่ง
  6. ส่ง QR PromptPay / เลขบัญชี / PayPal ให้ลูกค้า

ส่วนที่ 9 — ตรวจสอบการชำระเงิน (4 ขั้น)

  1. ตรวจสอบสลิปหรือการชำระผ่าน PromptPay / PayPal
  2. เช็คยอดเข้าใน Mobile Banking
  3. แจ้งลูกค้า "ยืนยันชำระเงินแล้ว" ทาง DM
  4. บันทึกออร์เดอร์ลงระบบ

ส่วนที่ 10 — แพ็คและจัดส่งสินค้า (5 ขั้น)

  1. แพ็คสินค้า + เขียนใบปะหน้าที่อยู่ลูกค้า
  2. ส่งพัสดุกับขนส่ง (Flash / Kerry / J&T / EMS)
  3. แจ้งเลข Tracking ให้ลูกค้าทาง DM
  4. ลูกค้าติดตามพัสดุผ่านแอปขนส่ง
  5. ลูกค้ารับสินค้าแล้ว → ขอรีวิวทาง DM

ส่วนที่ 11 — หลังการขายและ Retention (5 ขั้น)

  1. ขอรีวิวและรูปถ่ายจากลูกค้า
  2. Repost รีวิวลูกค้าจริง สร้าง Social Proof
  3. Follow up ลูกค้าหลัง 3–5 วัน ทาง DM
  4. วิเคราะห์ X Analytics: Impressions / Engagement / Link Clicks
  5. ปรับ Content Strategy: โพสต์ไหน Engagement สูง → ทำเพิ่ม
